Ejercicio de java script

Crea una página web con al menos tres párrafos de texto y dentro de ellos un número indeterminado de enlaces (al menos uno debe enlazar a google). Haciendo tan sólo uso del DOM muestra los siguientes datos: • Número de enlaces de la página • Dirección a la que enlaza el penúltimo enlace • Numero de enlaces que enlazan a google • Número de enlaces del tercer párrafo

Imagina que para una web que estás diseñando se te pide que en un apartado sólo se muestre una parte de un texto, y al pulsar un enlace “Ver Articulo Completo” se muestre el resto del texto. Implementa esta funcionalidad en una nueva página.

Estás diseñando una aplicación web para subir ficheros al servidor. Haciendo uso de DOM, consigue que el usuario pueda adjuntar tantos ficheros como desee. Por tanto inicialmente será necesario un input de tipo fichero, un botón “Adjuntar otro fichero” y un segundo botón “Enviar”, de modo que al pulsar el primero de los botones automáticamente aparezca otro input para elegir fichero.
